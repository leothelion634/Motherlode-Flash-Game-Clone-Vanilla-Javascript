# Motherlode-Flash-Game-Clone-Vanilla-Javascript
Motherlode-Flash-Game-Clone-Vanilla-Javascript


Please run index.html, this is vanilla Javascript game like Motherlode from XGenStudios flash game

You can move with arrow keys, you can dig down, left, or right, if you fly into the grey area with the names you must press space bar to enter them and use them

Dont run out of fuel! Go as deep as you can and make as much money as you can!

Have fun!
